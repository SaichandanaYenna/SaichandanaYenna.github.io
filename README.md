# Saichandana Yenna Portfolio

This repository contains a GitHub Pages portfolio for Saichandana Yenna.

## Site

When GitHub Pages is enabled, the site will be available at:

https://SaichandanaYenna.github.io

## Files

- `index.html` - portfolio content and page structure
- `styles.css` - responsive visual design
- `script.js` - mobile navigation and footer year
- `assets/` - place for resume, profile photo, diagrams, and screenshots

## Updating Content

Edit `index.html` to update:

- LinkedIn URL
- Project links
- Work experience
- Education
- Resume link, if a public resume is added

Avoid publishing confidential company data, private documents, credentials, or proprietary code.
